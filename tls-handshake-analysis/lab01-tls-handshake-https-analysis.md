# TLS Handshake Analysis

## Filter Used

tls

## Description

TLS (Transport Layer Security) is used to encrypt communication between a client and a server.

This protocol is used in HTTPS connections.

## Observations

When visiting secure websites such as wikipedia.org and instagram.com, TLS packets were observed during the handshake process.

This handshake establishes encryption parameters before the data exchange begins.

## Security Insight

TLS analysis helps identify:

- Encrypted communication
- Suspicious TLS certificates
- Potential command and control traffic
