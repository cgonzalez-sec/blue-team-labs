# Blue Team Labs

This repository documents my practical cybersecurity training focused on Blue Team skills and network traffic analysis.

The goal of these labs is to understand how real network communication works and how defenders analyze traffic to identify suspicious activity.

All exercises are performed in controlled lab environments.

• DNS Investigation  
• TCP Handshake Analysis  
• TLS Encrypted Traffic Inspection  
• Basic Network Behavior Analysis
